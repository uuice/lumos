动画技术说明

Intersection Observer API

当元素进入视口时触发动画，适用于滚动触发动画效果。页面加载时元素从底部淡入。

anime.js

强大的JavaScript动画库，提供丰富的动画效果和控制选项。页面加载后执行复杂的动画序列。

CSS Scroll-Driven Animations

现代CSS技术，使用 animation-timeline: view() 实现基于滚动的动画效果。
